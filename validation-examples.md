# AUREN LANGUAGE — VALIDATION EXAMPLES

## 1. PURPOSE

This document shows correct and incorrect usage examples of Auren.

It is used for learning and consistency checking.

---

## 2. VALID EXAMPLES

✔ VALID:

dor ara nora  
("I speak language")

ara dor nora  
(emphasis mode)

nora dor ara  
(poetic mode)

dor-a ara nora  
(present tense)

nor-dor ara nora  
(negation form)

ke dor ara nora?  
(question form)

vesh + thal → veshal  
(compound word)

---

## 3. INVALID EXAMPLES

❌ INVALID:

ara nora dor  
(if strict default structure is enforced)

dor xyz ara nora  
("xyz" is undefined root)

dor !!! nora ara  
(symbols not defined in system)

dor-abc ara nora  
(undefined morphological suffix)

---

## 4. AMBIGUITY RULE

Ambiguity is VALID only if:

- structure is correct
- roots are defined
- meaning can be interpreted consistently

Otherwise, it is INVALID.

---

## 5. EVOLUTION NOTE

Some INVALID forms may become VALID if:

- they become widely used
- they stabilize meaning
- they are integrated into system updates
