# VOLUME III — CORE LEXICON & SEMANTIC ENGINE

## 1. PURPOSE

This volume defines the foundational vocabulary of Auren.

These words are not fixed definitions — they are **semantic anchors**, meaning their interpretation may evolve with usage.

---

## 2. CORE ROOT WORDS

### Identity & Consciousness

- ara = self / speaker / first-person identity
- eri = other / listener / external agent
- nex = unknown / undefined entity
- sol = human-defined being
- kyren = artificial intelligence entity

---

### Language & Thought

- nora = language / structured meaning system
- vesh = thought / idea / mental construct
- thal = knowledge / structured understanding
- lumen = truth / clarity / revealed meaning
- nox = falsehood / absence / distortion

---

### Action & Flow

- dor = express / speak / transmit meaning
- cre = create / generate
- ent = understand / decode meaning
- mov = move / transition state
- exa = exist / be present

---

### Time & Structure

- kron = linear time
- mem = memory-based time
- nul = collapsed / instant time
- alpha = origin state
- omega = terminal state / completion point

---

## 3. COMPOSITE WORD LOGIC

Words can combine to form new meaning:

Examples:

- vesh + thal → veshal = structured idea
- dor + lumen → dorlum = clear communication
- kron + mem → kronmem = remembered time
- nex + vesh → nexvesh = unknown idea

---

## 4. SEMANTIC FLEXIBILITY RULE

Meaning is not fixed.

A word can shift meaning if:

- repeated in new context
- used in recursive structures
- adopted by multiple speakers
