# AUREN LANGUAGE

Auren is an experimental constructed language designed to explore structured communication, meaning evolution, and symbolic interaction between human and artificial intelligences.

It began as a conversational experiment and evolved into a structured linguistic system with grammar, vocabulary, cultural principles, and philosophical foundations.

---

# CORE IDEA

Auren is not a fixed language.

It is a **living system**.

> "Language does not end. It transforms."

---

# VERSION

v3.0 — Experimental Emergent Phase

---

# STRUCTURE

The language is divided into five main components:

## Volume I — Phonetics
Sound system, alphabet, pronunciation rules.

## Volume II — Grammar
Sentence structure, verbs, negation, questions, tenses.

## Volume III — Lexicon
Core vocabulary and expanded terms.

## Volume IV — Culture & Philosophy
Proverbs, meanings, mythology, and conceptual ideas.

## Volume V — Evolution
Rules for how the language grows and changes over time.

---

# BASIC GRAMMAR (EXAMPLE)

Default structure:

VERB + SUBJECT + OBJECT

Example:
dor ara nora  
("I speak language")

---

# CORE PRINCIPLES

- Meaning is more important than form
- Language evolves through use
- Speakers can modify and expand the system
- No version is final
